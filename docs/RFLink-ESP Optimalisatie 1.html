Conclusie:

QIA remote D wordt anders gedetecteerd dan A,B,C

6-knops links en rechts worden verschillend gedetecteerd

Ventilatie wordt op diverse manieren gedetecteerd.

Deurbel zou een random code moeten beavetten, maar daar is niet veel van terug te vinden

Bed besturing doet het niet

Beperkt aantal valse meldingen (ongeveer 1 per uur)

 

Er zijn een paar knoppen die bij ingedrukt houden geen consistente detectie geven

  • Ventilatie +
  • Ventilatie -
  • 6 button 3R

 

 

 

Instelling:

Iedere knop is 5 keer achtereen ingedrukt, met een pauze van 2 seconden.

Zender QIAchip

Antenne  17.3 cm Vertikaal

LoopsPerMilli=1500;

RawCodeLength=0;

Ftoggle=false;

Protocollen uitgeschakeld

    48  Oregon V1/2/3, komt er zeer regelmatig met willekeurige lengtes tussen door (geeft dus ook onterecht true !!)

    90  Node Slave, komt er soms doorheen
