SI4432:  QIA implementatie

Belangrijk:

  • Zeer goede ontkoppeling van de voedingsspanning
  • Meer dan 1 sequence in het FIFO stoppen en verzenden als 1 bericht (4 stuks past precies)
  • Het buffer moet als eerste een 0xFF bevatten (schrijf command naar 0x7F)

 

 

Besturing vanuit de Raspberry:

QIA_Rolluik_Achter = '1011 0000 0001 1010 1010'.replace(' ','')    # 20 bits address

QIA_Rolluik_Keuken = '1011 0000 0010 1010 1010'.replace(' ','')    # 20 bits address

QIA_Rolluik_Voor   = '1011 0000 0100 1010 1010'.replace(' ','')    # 20 bits address

 

Rolluik_Open  = '0001'

Rolluik_Dicht = '0010'

Rolluik_Uit   = '1000'

 

 

 

  SSPBUF  = [ 0xFF ]

  for i in range ( 4 ) :

    SSPBUF += Temp_Buffer

Temp_Buffer bevat

  # Address bits 20 = 10 bytes

  # ************************

  # Data bits 4 = 2 bytes

  # ************************

0 gaat als 8 nibble het buffer in

1 gaat als E nibble het buffer in
